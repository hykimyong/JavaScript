### 변수
### var (함수 스코프)
1. 선언 및 초기화(undefined 할당) 
2. 할당단계  

### let  (블록 스코프)
1. 선언 단계
2. 초기화 단계
3. 할당 단계

### const  (블록 스코프)
1. 선언 + 초기화 + 할당

### 함수 스코프 : 함수내에 선언된 변수  
### 블록 스코프 : 함수,if,for,while,try/catch문

### 호이스팅 : 스코프 내부 어디서든 변수 선언은 최상위에 선언된것 처럼행동
```
실질적으로 var,let,const모두 호이스팅은 일어나지만  
let과 const는 TDZ의 영향으로 할당하기전에는 사용할수 없음 (선언을 해야 사용 가능)
```
